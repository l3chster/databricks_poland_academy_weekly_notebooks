# Databricks Poland Academy - Weekly Notebooks

This repository contains laboratories and assignments completed as part of the Databricks Poland Academy program. The project focuses on implementing data engineering best practices using Databricks, PySpark, and Delta Lake architecture.

---

## 📁 Repository Structure

The project is organized into weekly laboratory directories. Each folder contains dedicated notebooks and tasks:

*   **`Lab_1/`** – Databricks Fundamentals & DEV Setup (Week 1) 
*   **`Lab_2/`** – Azure Services & Shared Lakehouse Setup (Week 2)
*   **`Lab_3/`** – Streaming & Incremental Ingestion (Week 3)
